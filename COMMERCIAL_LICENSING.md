# Guizang Skill 商業授權合作說明

這份說明適用於希望在自己的 AI 產品、Agent 平臺、創作工具、外掛市集或工作流系統中接入 Guizang Skill 能力的合作方。

Guizang Skill 系列的核心能力包括但不限於：根據正文語意自動規劃圖文卡片、呼叫真實相簿素材、處理截圖與網頁素材、生成地圖／路線元件、完成多比例社交圖片排版、生成高品質 PPT／簡報，以及對中文 AI 文字進行自然化改寫。

本說明中的價格為首批合作參考價，具體合作範圍、產品線、授權期限、適配深度、結算方式與交付內容，以雙方最終簽署的協議為準。

## 熱門專案介紹

目前較受歡迎、適合商業授權或平臺合作的專案主要包括以下三個。

### 1. Guizang Social Card Skill

專案位址：[op7418/guizang-social-card-skill](https://github.com/op7418/guizang-social-card-skill)

這個 Skill 面向 Instagram、Facebook、Threads、影片封面、社群媒體圖文卡片等場景。它可以根據正文內容自動拆解頁面結構，結合真實圖片、截圖、地圖、商品圖或網頁素材，生成多比例、多頁面的社交卡片。

適合接入的產品場景：

- AI 寫作工具中的一鍵配圖能力。
- Instagram / Facebook / Threads 內容創作工具。
- Agent 平臺中的圖文卡片生成 Skill。
- 影片工具中的封面生成能力。
- 企業內容行銷、產品說明、教學卡片生成場景。

核心價值：

- 減少使用者手動找圖、裁圖、排版的成本。
- 避免純色塊、emoji 或 AI 插畫導致的廉價感。
- 支援真實相簿、截圖、地圖等證據型視覺素材。
- 輸出可以直接用於社交平臺釋出的成品圖。

### 2. Guizang PPT Skill

專案位址：[op7418/guizang-ppt-skill](https://github.com/op7418/guizang-ppt-skill)

這個 Skill 面向 PPT、簡報、匯報材料、課程教材、商業提案與視覺化表達場景。它能夠根據使用者提供的主題、文案、資料或結構，生成具有明確視覺風格的簡報頁面。

適合接入的產品場景：

- AI PPT 生成工具。
- 辦公套件、線上檔案、知識庫與協作平臺。
- 企業匯報、教育訓練、公開演講、課程內容生成。
- Agent 平臺中的 PPT 生成或美化能力。
- 將文章、檔案、會議紀要轉換為簡報的工作流。

核心價值：

- 幫使用者從空白頁直接進入可用的視覺表達。
- 提供比一般模板更強的排版、風格與敘事能力。
- 適合與檔案、知識庫、會議紀要、商業分析等工具結合。
- 可以作為 AI 辦公產品的重要差異化能力。

### 3. Humanizer-zh

專案位址：[op7418/Humanizer-zh](https://github.com/op7418/Humanizer-zh)

這個 Skill 面向中文文字自然化、去除 AI 味、內容潤色與表達最佳化場景。它可以識別並修復常見的 AI 寫作痕跡，讓中文內容更自然、更像真人書寫。

適合接入的產品場景：

- AI 寫作工具中的「一鍵去 AI 味」功能。
- 內容平臺、Facebook、Instagram、Threads、影片指令碼等創作工具。
- 履歷、Email、商業文案、社群文案潤色。
- 教育、辦公、行銷、知識管理類產品。
- Agent 平臺中的中文文字最佳化 Skill。

核心價值：

- 提升 AI 生成文字的可讀性與真實感。
- 降低使用者後期人工潤色成本。
- 適合放在生成鏈路的最後一步，作為文字品管與風格修正能力。
- 對中文場景更友善，不只是簡單改寫或同義詞替換。

以上三個專案可以單獨授權，也可以組合授權。組合授權適合希望一次性補齊「寫作、配圖、PPT、潤色」完整內容生產鏈路的平臺。

## 合作等級

### 1. 深度內建授權

這是最高階別的合作方式，適合希望將 Guizang Skill 的能力深度整合到自有產品中的團隊。

合作方可以將相關能力內建到自己的產品流程中，也可以選擇不在產品介面中展示 Guizang Skill 的署名。經雙方確認後，合作方可以對外將該能力作為自身產品能力的一部分進行介紹。

我會配合合作方完成必要的產品適配、效果最佳化、提示詞與工作流調整，使該能力在合作方產品中達到較好的使用體驗。必要時，我也可以配合合作方進行聯合宣傳或案例釋出。

參考費用：

- 新創公司、早期團隊或熟人合作：一次性授權費用為 10 萬元起。
- 日活 10 萬級以上產品：一次性授權費用為 30 萬元起。

預設包含：

- 授權一個明確產品或一個明確產品線使用。
- 允許深度內建到合作方產品中。
- 允許不展示 Guizang Skill 署名。
- 提供基礎接入建議與一定範圍內的效果最佳化配合。
- 可根據實際情況配合對外宣傳。

預設不包含：

- 原始碼或智慧財產權轉讓。
- 獨家授權或品類排他。
- 跨多個產品線、多個品牌或多個客戶的轉授權。
- 長期駐場開發、持續客製化開發或無限期維護。
- 將 Guizang Skill 能力包裝後再出售給第三方平臺。

如果合作方需要獨家授權、區域排他、品類排他、原始碼買斷、長期深度客製化、白牌轉售或多產品線使用，需要單獨報價與簽署補充協議。

### 2. 上架與曝光合作

這是標準合作方式，適合希望在自己的平臺中上架 Guizang Skill，但不需要深度客製化或買斷能力的合作方。

合作方可以在自己的平臺、外掛市集、Agent 市集或工作流市集中上架 Guizang Skill，並允許使用者透過平臺呼叫相關能力。我會保留 Guizang Skill 的版權與署名，不將其視為合作方自研能力。

參考費用：

- 固定合作費用：3 萬元。

這筆費用也可以視為對 Guizang Skill 專案的贊助。作為回報，我會在專案 README、Skill 檔案或相關說明中曝光合作方品牌，具體曝光位置與方式由雙方確認。

預設包含：

- 授權合作方在一個平臺或一個外掛市集中上架 Guizang Skill。
- 保留 Guizang Skill 名稱、版權與署名。
- 提供基礎上架支援。
- 提供基礎版本更新，但不承諾深度適配。
- 在 Guizang Skill 專案檔案中曝光合作方品牌。

預設不包含：

- 去署名或白牌使用。
- 宣稱該能力為合作方自研。
- 深度產品客製化、私有化重構或長期專項維護。
- 獨家授權、排他授權或轉授權。

### 3. 收益分潤模式

這是最低門檻的合作方式，適合已經具備付費能力的平臺，例如使用者可以透過點數、Token、額度包、訂閱會員或按次付費來使用 Skills 的平臺。

合作方可以將 Guizang Skill 上架到平臺中，由使用者按平臺規則付費使用。合作方需要定期向我結算由 Guizang Skill 產生的收益分潤。

參考方式：

- 不收取固定上架費用，或僅收取很低的接入費用。
- 合作方按月或按季結算收益分潤。
- 分潤比例根據平臺計費方式、流量規模、推廣資源與技術支援成本另行確認。

預設包含：

- 授權合作方在平臺中上架 Guizang Skill。
- 使用者可以透過平臺的點數、Token、額度或付費系統呼叫該能力。
- 我不提供額外深度適配，只維護通用版本能力。

預設要求：

- 合作方需要提供清晰的呼叫資料、收入資料與結算週期。
- 合作方不得隱藏、混淆或挪用 Guizang Skill 產生的實際收益。
- 合作方需要在合理位置保留 Guizang Skill 名稱或來源說明。
- 如果後續需要深度適配、去署名、白牌或對外包裝為自研能力，應升級為深度內建授權。

## 通用授權邊界

除非雙方另有書面約定，以上任一合作等級均不代表：

- 轉讓 Guizang Skill 的版權、商標、名稱、原始碼或其他智慧財產權。
- 授予合作方獨家使用權。
- 允許合作方將 Guizang Skill 再授權、轉售或封裝給第三方。
- 允許合作方使用 Guizang Skill 的內容、程式碼、提示詞、工作流或生成結果訓練競品模型或複刻類似產品。
- 允許合作方移除版權資訊後，將 Guizang Skill 作為完全自有資產再次商業化。

如需上述權利，需要單獨協商並簽署獨立協議。

## 合作流程

1. 合作方說明產品型態、接入場景、預計使用者規模與商業模式。
2. 雙方確認合作等級：深度內建授權、上架與曝光合作，或收益分潤模式。
3. 雙方確認授權範圍，包括產品線、平臺、使用方式、是否署名、是否需要適配、是否需要宣傳配合。
4. 雙方確認費用、付款方式、結算週期與交付內容。
5. 簽署協議並完成付款或結算設定。
6. 我提供對應等級的接入、上架或適配支援。

## 備註

以上價格為目前階段的合作參考價。隨著 Guizang Skill 的使用者規模、Star 數、案例數量與平臺影響力提升，後續商業授權價格可能調整。

如果合作方希望盡快接入，建議優先選擇首批合作方案。首批合作方可以獲得更低的授權價格、更靈活的合作方式，以及更直接的產品適配溝通。

本說明僅作為商業合作框架，不構成最終法律協議。正式合作前，雙方可以根據實際需求補充合約條款。
